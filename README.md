<h3>Esse projeto simula um piano que reproduz os sons reais das teclas conforme suas respectivas notas </h3>
<ul>
  <li>Volume range funcional</li>
  <li>Toggle de teclas ->  Permitindo visualização das letras do teclado para cada tecla do piano </li>
</ul>

<br>

<h3>Conhecimentos adiquiridos nesse projeto:</h3>
<br>

<ul>
  <li>Manipulação de DOM </li>
  <li>Capturar elementos HTML </li>
  <li>adicionar e Eemover classes dinamicamente</li>
  <li>Toggle com classe</li>
  <li>Arrow Function</li>
  <li>Eventos</li>
  <li>Edição elementos HTML com CSS</li>
  <li>Pseudo Elementos CSS</li>
</ul>
